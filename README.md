# Tic Tac Toe

A web based Tic Tac Toe game integrated with an AI.

> [*Play here!*](https://vytdev.github.io/tictactoe)

This features:

- Easy to play
- Interactive
- Game scoring
- 1 on 1
- Easily customizable
- Computer mode
- Smart mode (for computer)
